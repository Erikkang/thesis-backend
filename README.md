Skin Analyzer Backend
====================

1. Download the "models" folder from Google Drive and place it in the project root.
   link: https://drive.google.com/drive/folders/1RX8JQQArUccn1cPlXokXmukIT5QF16Tc?usp=sharing

3. Create a virtual environment:
   python -m venv venv

4. Activate the virtual environment:
   Terminal:
     venv\Scripts\activate
   Mac/Linux:
     source venv/bin/activate

5. Install dependencies:
   pip install -r requirements.txt

6. Run backend:
   python app.py

Note:
- Make sure the "models" folder is in the root before running the app.
- Do not upload "venv" or "models" to GitHub.
